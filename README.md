# Helpdesk Lab

A hands-on helpdesk lab built to gain real-world experience with the tools, workflows, and troubleshooting procedures used in enterprise IT support environments. As someone looking to break into a helpdesk role, I built this repository to go beyond theory and actually practice the scenarios I would encounter on the job — from account lockouts to Group Policy issues — documented the way a real technician would.

All scenarios in this lab are worked as real tickets using **Jira** as the ticketing system, following a complete ticket lifecycle from acknowledgment to resolution and closure.

---

## Environment

| Component | Details |
|---|---|
| **Domain Controller** | Windows Server 2022 |
| **Client Machine** | Windows 10 Pro — CLIENT01 |
| **Domain** | homelab.local |
| **Hypervisor** | Oracle VirtualBox |
| **Ticketing System** | Jira (Kanban — Justins-Helpdesk) |

---

## Skills Demonstrated

- Active Directory user account management
- Group Policy troubleshooting
- Ticket lifecycle management in Jira
- Identity verification procedures
- Helpdesk escalation procedures
- IT documentation and SOP writing
